# Book My Walks

The all-in-one platform for independent dog walkers and pet service entrepreneurs.

**Live site:** https://bookmywalks.com

## Stack
- GitHub Pages (frontend hosting)
- Supabase (backend, auth, subdomain routing — coming soon)

## Pages
- `index.html` — Homepage
- `demo.html` — Booking demo
- `build.html` — Build your page
- `signup.html` — Walker signup
- `walker.html` — Walker dashboard
